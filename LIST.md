# V2 Chapters list

## The "map" to understand this

> "[]" (blank) means that this chapter has not been implemented yet
> "[🟨]" means that this chapter has been half-implemented, and still needs some work
> "[🟩]" means that this chapter has been implemented and it's working (interpretation only)
> "[🟦]" means that this chapter has been implemented and it's working (compilation and interpretation)

> ⚠️ **Accuracy note (2026-07-02):** the `[🟦]` marks below reflect that each chapter *parses and
> runs*, but many `std.*` chapters are **typed stubs**, not real implementations (e.g. networking,
> databases, GUI, hardware, media, ML). For the authoritative implemented/partial/stub status, see
> `NOT_YET_IMPLEMENTED.md`. The core language and pure-computation stdlib are genuinely `[🟦]`.
